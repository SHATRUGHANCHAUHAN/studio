# **App Name**: FridgeChef

## Core Features:

- Ingredient Input: Allow users to input the ingredients they have available.
- Recipe Generation: Generate recipes based on the provided ingredients using a Large Language Model tool to consider common ingredient pairings and recipe databases.
- Recipe Display: Display the generated recipes with a title, ingredients list, and cooking instructions.
- Additional Information: Fetch from an external api nutritional data about each recipe. Display the nutritional facts as well as preparation time for the selected recipe.

## Style Guidelines:

- Primary color: Fresh green (#90EE90) to reflect the natural ingredients and healthy cooking.
- Background color: Light pale green (#F0FFF0), nearly white to provide a clean backdrop.
- Accent color: A slightly darker, contrasting green (#3CB371) to highlight interactive elements and important information.
- Font: 'PT Sans', a humanist sans-serif font suitable for both headlines and body text providing a clean and readable interface.
- Use simple, outline-style icons to represent different ingredients and cooking steps.
- Maintain a clean and intuitive layout with clear sections for ingredient input, recipe display, and additional information.
- Use subtle transitions and animations to provide feedback when generating recipes and interacting with the app.
